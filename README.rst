``project``
-----------

Deploy a self-contained standalone salt minion to manage locally a project on
the minion.

Available states
~~~~~~~~~~~~~~~~

``project``
+++++++++++


Deploys all project.

Available modules
~~~~~~~~~~~~~~~~~


``project.call``
++++++++++++++++

Call a specific action on the project's salt setup.
